


## 技术栈

- React栈：React + React-Router + Redux + React-Redux + Redux-Saga + Webpack
- 脚手架：umi + dva
- UI组件库：无
- 测试：Jest + enzyme
- 其他
	- 当前版本暂未实现 TypeScript
	- 当前版本暂未实现 SSR 功能

## 运行

访问在线demo：[https://hzbdesigner.github.io/exercise](https://hzbdesigner.github.io/exercise)


本地运行
```
npm install
npm start
```

单元测试
```
npm test
```

打包发布到github gh-pages
```
npm run build
npm run deploy-github
```
